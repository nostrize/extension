Nostr Settings

Store your password in a password manager or keep it somewhere safe.

For public key, use our GitHub Connect project to generate a NIP-05 address and put the pubkey in its nostr.json.

For private/secret key, we will send the encrypted password to our relay. The extension will fetch the public key from the GitHub Connect URL, query our relay for the encrypted private key, and finally, the user will use the password to decrypt. The extension will ask for the password whenever it needs to sign an event.
